# Reconnaissance faciale — SVM

Application de reconnaissance faciale avec un modèle SVM (scikit-learn), interface Streamlit.

## Lancer en local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Déploiement

Les modèles (`.joblib`) sont inclus. Pour déployer sur [Streamlit Community Cloud](https://share.streamlit.io), connecter ce dépôt et indiquer `app.py` comme fichier principal.
